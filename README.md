# docs-editor-mern
## Description
    A Rich Text Editor With Cloud-based Syncing Features
## Technologies Used
    ReactJS
    NodeJS
    ExpressJS
    MongoDB & Mongoose
## How to run
    git clone {repo-link}
    cd into repo/client
    npm install
    npm run dev
    
    cd into repo/server
    npm run dev
    Open http://localhost:5173 to view it in your browser.
